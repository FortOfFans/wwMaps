
# Wuthering Waves: JSON data

This repository contains game data for Wuthering Waves for all kinds of stuff: Characters, Weaopons, etc...

> Game Version: 1.0.2

## API Reference

#### Get data

```http
  https://raw.githubusercontent.com/FortOfFans/wwMaps/master/{lang}/{type}.json
```

> Types:
> - avatar
> - weapons
> - relics
> - items
> - monster
> - curio
> - blessings
